===========================
How to use 2-matrix_divided
===========================

This module divide a matrix by a coefficient

Usage
=====
``matrix_divided(matrix, div) return a matrix whith new values

>>> matrix_divided = __import__('2-matrix_divided').matrix_divided
>>> matrix = [[1,2,3], [4,5,6]]
>>> matrix_divided(matrix, 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

The value are rounded to 2 decimal places
